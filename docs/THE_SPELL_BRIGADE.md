# The Spell Brigade Module

## Status

Supported module scaffold.

## Purpose

The Spell Brigade module provides a local-first workspace for co-op run planning and review. It reuses the shared Gaming chat and screenshot infrastructure while keeping game-specific planning organized around the game's wizard, spell, upgrade, synergy, and objective-driven run loop.

Select `The Spell Brigade` from the game picker at the start of the Gaming workspace toolbar. The existing expandable Gaming navigation tree remains an alternate selection path.

## Current Sections

| Section | Purpose |
| --- | --- |
| Home | Overview of the module's planning areas. |
| Chats | Persisted game-scoped planning conversations. |
| Wizards | Future wizard, starter-spell, role, and team-composition notes. |
| Spells | Future spell, element, behavior, and upgrade-path notes. |
| Upgrades | Future progression and unlock-priority notes. |
| Synergies | Future spell-combination and co-op team-build notes. |
| Runs | Future objective, outcome, build-result, and lessons-learned records. |
| Screenshots | Shared local Gaming screenshot capture and review. |

## Scope Boundary

Current scope is the seeded game definition, module navigation, scaffolded planning sections, game-scoped chat, and shared screenshots.

Persisted wizard catalogs, spell catalogs, upgrades, synergies, run history, build recommendations, automatic game-data imports, and live game integration remain out of scope until explicitly requested.
