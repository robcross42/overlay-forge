# Local Reference DLLs

Copy GearBlocks/BepInEx reference DLLs here only in the ignored working copy under `gearblocks-bepinex-workspace`.

The direct plugin preview renderer needs `UnityEngine.ImageConversionModule.dll` in addition to the existing Unity, BepInEx, IL2CPP, and GearBlocks references so it can encode PNG captures.

Do not commit DLLs from:

- GearBlocks
- BepInEx
- Unity
- third-party mods
